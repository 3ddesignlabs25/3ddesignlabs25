Arduino Nano / ESP32 Wi-Fi Alarm

A compact embedded system that auto-connects to Wi-Fi and hosts a web server for alarm control, configuration, and system status. Includes physical buttons and LED indicators for offline operation.

The enclosure is a precision-fit PETG 3D-printed shell with custom buttons and LED diffusers.

    int timeAttempts = 0;
    while (time(nullptr) < 100000 && timeAttempts < 20) {
        delay(500);
        Serial.print(".");
        timeAttempts++;
    }

    if (time(nullptr) > 100000) {
        serialPrint("Time synced!");
        setupWebServer();
    } else {
        serialPrint("Time sync failed.");
    }
} else {
    serialPrint("WiFi connection failed. Retrying...");
}
